컴퓨터의 두뇌 CPU

CPU는 무엇이고 어떤 동작을 할까?



CPU는 3가지의 동작을 수행합니다.
1. 데이터 가져오기(Fetch) :
- 데이터를 RAM으로부터 가져와 프로그램 카운터(PC)에 데이터(명령어)를 보관
2. 디코딩(Decoding) :
- 명령을 가져와서 명령 해독기라는 회로로 명령을 전달
3. 실행(Execute) :
- 마지막 단계에서 디코딩된 명령문은 완료될 CPU의 관련 부분으로 전송

그렇기에 램을 구매시에는 빠르게 램을 읽어와 전달 할 수 있는지의 계산 척도인
CPU 성능 = (클럭 * IPC) * 코어 수를 기억하여
클럭, IPC, 코어 수가 많은 램을 구매하면 됩니다.
